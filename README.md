
# es-big-screen 大屏可视化项目模板

这是一个基于Vue 3、Echarts、高德地图和Pinia开发的大屏可视化项目，提供了如下功能

*   大屏适配
*   图表组件(Echarts)封装
*   高德地图组件封装
*   拖拽布局
*   入场动画
*   无缝滚动
